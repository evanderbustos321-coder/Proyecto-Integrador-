# Proyecto-Integrador-
Nombre del proyecto: El impacto de las nuevas tecnologías en la sociedad: desarrollo y proyección de soluciones informáticas

Nombre: Evander Bustos


Fecha: 28/06/26


Materia: Logica de Programacion


Objetivo del sistema:
Desarrollar un juego de Piedra, Papel o Tijera en Python que permita al usuario jugar contra la computadora aplicando variables, funciones, condicionales, ciclos y validación de datos.


Funcionalidades:
- Menú principal.
- Jugar contra la computadora.
- Elección aleatoria de la computadora.
- Mostrar el ganador.
- Ver el marcador.
- Salir del program
